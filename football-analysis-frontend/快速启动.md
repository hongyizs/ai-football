# 🚀 快速启动指南

## 前提条件

✅ Node.js 16+ 已安装
✅ pnpm 已安装
✅ 后端服务已启动（http://localhost:9000）

## 启动步骤

### 1. 安装依赖（首次运行）

```bash
cd football-analysis-frontend
pnpm install
```

### 2. 启动开发服务器

```bash
pnpm dev
```

### 3. 访问应用

打开浏览器访问：**http://localhost:5173**

### 4. 登录系统

- **用户名：** admin
- **密码：** 123456

## 故障排除

### 问题1：跨域错误
**解决方案：**
1. 确保后端已启动
2. 检查`.env.development`配置
3. 重启前端服务

### 问题2：WebSocket连接失败
**解决方案：**
1. 检查后端WebSocket服务
2. 确认token有效
3. 查看浏览器控制台错误

### 问题3：登录失败
**解决方案：**
1. 确认用户名密码正确
2. 检查后端认证服务
3. 清除浏览器缓存

### 问题4：数据不显示
**解决方案：**
1. 检查网络请求
2. 查看控制台错误
3. 确认后端API正常

## 项目结构

```
football-analysis-frontend/
├── src/
│   ├── components/          # 组件
│   │   ├── AnalysisTable.tsx       # 表格组件

│   │   ├── AnalysisList.tsx        # 主列表
│   │   ├── Login.tsx               # 登录
│   │   └── ErrorBoundary.tsx       # 错误边界
│   ├── services/            # 服务层
│   │   ├── api.ts          # API服务
│   │   ├── auth.ts         # 认证服务
│   │   └── websocket.ts    # WebSocket服务
│   ├── store/              # 状态管理
│   │   └── analysisStore.ts
│   ├── types/              # 类型定义
│   │   └── index.ts
│   ├── App.tsx             # 根组件
│   ├── index.tsx           # 入口文件
│   └── index.css           # 全局样式
├── public/                 # 静态资源
├── .env.development        # 开发环境配置
├── .env.production         # 生产环境配置
├── vite.config.ts          # Vite配置
├── tsconfig.json           # TypeScript配置
├── uno.config.ts           # UnoCSS配置
└── package.json            # 项目配置
```

## 环境变量

### 开发环境（.env.development）
```env
VITE_API_URL=/foot
VITE_WS_URL=ws://localhost:9000/foot/ws/analysis
```

### 生产环境（.env.production）
